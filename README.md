# PdfTangler

**PdfTangler** is a simple, fast, privacy-first **React-based PDF utility web app**.

It lets users **merge PDFs** and **reorder PDF pages** ‚Äî entirely **in the browser**.

‚ùó PdfTangler is **NOT** a Python tool, **NOT** a Streamlit app, and **NOT** a PDF data-extraction library.  

PdfTangler is a **frontend-only React application** web application for basic PDF utilities.
It runs fully in the browser and is NOT a Python or Streamlit tool.

Live demo: https://pdftangler.pages.dev
---

##  Privacy First by Design

- No servers
- No file uploads
- No login
- No tracking
- No hidden limits


All PDF processing happens **client-side**, so your files **never leave your device**.

---
##  Features

-  Merge multiple PDFs in any custom order
-  Reorder pages inside a PDF
-  100% client-side processing
-  Fast and lightweight
-  No login, no tracking, no friction
-  Works fully in the browser

---

##  Tech Stack

- **React.js**
